Meine Skills.

HTML

HTML (Hypertext Markup Language) ist das Grundgerüst jeder Webseite. Es hilft, Inhalte wie Texte, Bilder und Links so anzuordnen, dass sie im Browser richtig angezeigt werden. Mit HTML kannst du ganz einfach eigene Webseiten erstellen und sie strukturieren – egal, ob du eine einfache Seite oder eine komplexe Plattform bauen möchtest. Ohne HTML würde das Internet, wie wir es kennen, nicht existieren!

CSS

CSS (Cascading Style Sheets) gibt deiner Webseite das Aussehen! Während HTML die Struktur bereitstellt, sorgt CSS dafür, dass alles schön aussieht – von Farben über Schriftarten bis hin zu Layouts. Mit CSS kannst du deine Seiten individuell gestalten, sie ansprechend und benutzerfreundlich machen. Es ermöglicht dir, das Design flexibel zu ändern, ohne die Struktur deines HTML-Codes anzufassen. Kurz gesagt: CSS bringt Stil und Leben in deine Website!

Oben rechts schwebt ein Button mit. Mit diesem kannst du CSS deaktivieren und aktieren und hast somit einen direkten Eindruck, wie mächtig es sein kann.

JavaScript

JavaScript bringt deine Webseite in Bewegung! Es sorgt dafür, dass deine Seite interaktiv wird. Mit JavaScript kannst du dynamische Inhalte erstellen, auf Benutzereingaben reagieren und Animationen hinzufügen. Es macht deine Webseite lebendig, indem es Funktionen wie Schaltflächen, Formulare und Menüs interaktiv gestaltet. Mit JavaScript kannst du die Benutzererfahrung auf deiner Website deutlich verbessern!

Der Button für CSS funktioniert nur mithilfe von JS!
